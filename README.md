# Internet Provider Decision Support System

A web-based Decision Support System for selecting the best internet provider using a hybrid Multi-Criteria Decision Making approach with **MPE**, **SAW**, and **KMKK** methods, combined with **Google Play review sentiment analysis**.

## Technologies

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
- Google Play Scraper

##  Decision Methods

- MPE (Metode Perbandingan Eksponensial)
- SAW (Simple Additive Weighting)
- KMKK (Kompensasi Multi Kriteria)

##  Features

- Provider recommendation
- Provider ranking
- CRUD provider data
- Dashboard visualization
- Google Play sentiment analysis
- Provider catalog
